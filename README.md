# Catholic Liturgy Skills for Claude

A collection of Claude skills for generating professionally formatted Catholic liturgical documents.

## Available Skills

| Skill | Description |
|-------|-------------|
| [penitential-act-formatter](./penitential-act-formatter/) | Generates Penitential Act (Form C) documents with proper "say the black, do the red" formatting |
| [penitential-act-invocations](./penitential-act-invocations/) | Writing assistant that helps to write penitential acts that go with the readings |

## Installation

### For Claude Desktop / Claude Code

1. Download the `.skill` file for the skill you want from the [Releases](../../releases) page
2. Upload the skill file to Claude

### Manual Installation

Copy the desired skill folder to your Claude skills directory.

## Skills Overview

### Penitential Act

Generates formatted Penitential Act documents (.docx) for Mass worship aids.

**Features:**
- Red rubrics (Priest/Deacon labels) and black spoken text
- Sabon Next LT font
- Automatic font sizing to fit one page
- Centered title with Sunday, Season, and Year

**Output example:** `Penitential Act_1st Sunday of Advent_A.docx`

[View full documentation](./penitential-act/SKILL.md)
